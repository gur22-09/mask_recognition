# Open Source Mask Detection System [Model Trained with 150 Images]



## Setup
`git clone` the repo and `cd` into it by running the following command:

```bash
git clone <URL>
cd <Cloned Directory>
```

### `npm install`


## Run the App
### `npm start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
